Entity Name Recognizer Model

ระบบรู้จำ Named Entity นั้น ประกอบด้วยการระบุตำแหน่งและขอบเขตของชื่อ พร้อมทั้งระบุประเภทของชื่อดังกล่าว โดยจะมีการกำหนดชนิดของ Named Entity ที่จะรู้จำไว้ก่อน เช่น ชื่อคน ชื่อองค์กร ชื่อสถานที่ วันที่/เวลา เป็นต้น วิธีการในการรู้จำ Named Entity นั้น มีด้วยกันหลายวิธี เช่น การใช้พจนานุกรม การใช้กฎ การสร้างแบบจำลองทางสถิติด้วยหลักการ Machine Learning หรือเป็นการนำเอาทุกวิธีมาใช้ร่วมกัน ซึ่งต้องใช้ความรู้ทั้งในทางคอมพิวเตอร์และภาษาศาสตร์ แม้การรู้จำ Named Entity จะมีความสำคัญในการประมวลผลภาษาธรรมชาติ แต่ปัจจุบัน ยังไม่มีซอฟต์แวร์สำหรับการรู้จำ Named Entity ในภาษาไทยที่ผู้ใช้สามารถใช้งาน รวมทั้งสามารถฝึกฝนระบบด้วยชุดเอกสารของตัวเองได้ง่าย ด้วยเหตุนี้ โครงงานนี้จึงมีวัตถุประสงค์เพื่อพัฒนาซอฟต์แวร์เครื่องมือที่ผู้ใช้สามารถสร้างแบบจำลองเพื่อรู้จำ Named Entity ในโดเมนและประเภทของชื่อที่ตัวเองต้องการ ในการสร้างแบบจำลองเพื่อรู้จำ Named Entity สามารถทำได้โดยการนำเข้าคลังเอกสารสำหรับการฝึกฝนระบบ รวมทั้งฐานความรู้ทางภาษาอื่นๆ เช่น พจนานุกรม หรือชุดคำศัพท์ต่างๆ จากนั้นกำหนดค่าทางเลือก (option) และค่าพารามิเตอร์ต่างๆ ซึ่งทำให้ผู้ใช้สามารถพัฒนาระบบรู้จำ Named Entity ได้โดยไม่จำเป็นต้องมีความรู้ทางด้านการเขียนโปรแกรม รวมทั้งไม่จำเป็นต้องรู้ขั้นตอนวิธีการในการสร้างแบบจำลอง โดยระบบรู้จำ Name entity จะเป็นเป็นส่วนย่อยของการสกัดข้อมูลที่จะระบุและแยกประเภทของชื่อเฉพาะในข้อความที่กำหนดไว้ล่วงหน้า เช่น ชื่อคน, ชื่อองค์กร, ชื่อสถาที่, ปริมาณ และสกุลเงิน เป็นต้น
